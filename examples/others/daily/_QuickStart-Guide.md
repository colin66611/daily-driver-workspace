# 🚀 Daily Driver 快速上手指南

## 基本命令

### 1. 启动今天的工作
```
你：开启今天的工作

Agent：今天的能量状态如何？[高 / 中 / 低]

你：中

Agent：🅿️ 想法停车场内容：
       • 学习新技术（中优先级）
       • 整理读书笔记（低优先级）

       请告诉我今天想处理哪些事项？

你：处理学习新技术

Agent：[生成任务提案...]
       ✅ 今日工作已锁定。准备开始工作。
```

### 2. 展开任务
```
你：展开 M1

Agent：📋 当前任务：M1 - 学习新技术
       🎯 目标：掌握使用方法并应用到工作中
       🔒 规格已锁定

       [开始执行任务...]
       [实时更新 progress.md 和 findings.md]
```

### 3. 记录发现
在任务执行过程中，Agent 会自动记录：
- 关键发现 → `findings.md`
- 执行进展 → `progress.md`
- 错误记录 → `task_plan.md` 中的错误表格

### 4. 完成任务
```
你：完成

Agent：🎉 任务 M1 已完成！

       📊 总结：
       - 完成标准：✅ 全部达成
       - 关键发现：3 个
       - 遇到错误：0 次

       下一步：应用新技术到实际项目
```

### 5. 结束一天
```
你：结束今天的工作

Agent：📅 今日工作总结

       ✅ 已完成：
       - M1: 学习新技术

       ⏸️ 进行中：
       - 无

       📝 新想法：
       - 可以整理今天的学习笔记

       是否归档已完成的任务？[是 / 否]
```

## 文件命名规范

### 日期目录
```
YYYY-MM-DD/
例如：2026-03-30/
```

### 任务目录
```
task_M{数字}_{任务简名}/
例如：task_M1_学习新技术/
```

### 任务文件
每个任务目录包含三个文件：
- `task_plan.md` - 任务计划
- `progress.md` - 进展日志
- `findings.md` - 研究发现

---

## 💡 使用技巧

### 1. 随时保存
任何时候都可以说"保存"，Agent 会立即保存当前进展。

### 2. 规格锁定
任务规格一旦确认，执行过程中不可更改。如需调整，建议新建任务。

### 3. 2-Action 规则
每执行 2 个操作后，Agent 会自动保存发现并提醒。

### 4. 中断恢复
如果对话中断，下次开始时 Agent 会从文件恢复状态，不会丢失进度。

### 5. 想法停车场
随时记录新想法，避免打断当前工作。每天早上查看并选择今天要处理的。
