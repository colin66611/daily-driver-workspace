# Daily Driver Skill 评估标准 (简化版)

## 核心评估指标

### 1. 文件创建准确性
- **标准**: 必须创建正确的文件和文件夹
- **验证**: 检查路径和内容是否符合预期

### 2. 流程完整性
- **标准**: 从启动到完成必须完整闭环
- **验证**: 每个阶段是否都有明确的输出

### 3. 状态正确性
- **标准**: 状态标记必须准确(⏸进行中 → ✅完成)
- **验证**: 检查task_plan.md的状态字段

### 4. 用户指引清晰度
- **标准**: 每步操作后要有清晰的下一步指引
- **验证**: 输出是否包含明确的提示或确认
